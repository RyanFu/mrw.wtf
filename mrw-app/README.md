# Mobile app for [mrw.wtf](https://mrw.wtf)

## Usage

Install dependencies:

```bash
npm install -g react-native-cli
npm install
```

Fill config:

```bash
cp config.js{.template,}
```

Run packager:

```bash
npm start
```

Run app on iOS or Android:

```bash
react-native run-android
react-native run-ios
```
